# WorkoutGamifier
This is a Mobile-first web application where you can add and spend points to get random workouts from a pool of your choosing.

As a user, you can create the following: 
- Sessions, which represents a session of activity, where you mix workout with something else (like gaming, for example)
- Workout pools, which are groups of workout you've selected from a wider catalog
- Actions, which are irl actions for which you can get points. They are simply a pair of a description with a point reward (e.g. capture a pokemon: 1 point)

When starting a session, you can select a workout pool. During your session, you can add points for completed irl actions, and then, you spend points to receive a random workout from your workout pool.
